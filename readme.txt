== Description ==

Gestiona los clicks de las publicidades de tu sito, colocandole un simple link que redirecciona a la web del 
cliente pero reconoce los clicks que se realicen en la publicidades que coloques en tu sitio.


== Installation ==

1. Upload `publicidad` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Clients (Name and URL for redireccion).
4. Coloca a cada publicidad que tu cliente en el href del vinculo http://TUDOMINIO.COM/wp-content/plugins/publicidad/redirect.php?cliente=ID_DEL_CLIENTE_CARGADO
5. El ID del cliente cargado lo encuentras en la columna ID de los clientes que hayas cargado, la redireccion se realizara a la web que hayas colocado en la columna
redireccion cuando cargaste al cliente.
